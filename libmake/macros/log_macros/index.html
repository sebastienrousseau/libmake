The log_macros module contains macros related to logging messages at various log levels and formats. This module contains macros related to logging messages at various log levels and formats.

It includes a custom logging macro, macro_log_info, which allows logging messages with specified log levels, components, descriptions, and formats.

§Custom Logging Macro

The macro_log_info macro is designed for logging messages with customizable log levels, components, descriptions, and formats. It provides flexibility in defining log messages according to specific requirements.

§Parameters

  • $level: The log level of the message.
  • $component: The component where the log is coming from.
  • $description: A description of the log message.
  • $format: The format of the log message.
