Replaces placeholders in a template file with values from a set of parameters

Arguments

  • template_file - A PathBuf representing the path to the input template file
  • output_file - A PathBuf representing the path to the output file that will be created
  • params - A reference to a FileGenerationParams struct containing the values to be substituted into the placeholders
